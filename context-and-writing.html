Picture context like this:

─(pre)──<subject>──(post)─
  │                  │
  └───── context ────┘

Context surrounds the subject matter. So sententces are like functions:

              ┌─────────┐
─(precontext)→│ Subject │─(postcontext)→
              └─────────┘
